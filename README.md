# 🔐 PDF Password Protector

🚀 **Day 18/30 – 30 Days of Python Project Challenge**

A Python-based tool to secure PDF files by adding password protection and restricting permissions using `pikepdf`.

---

## 📌 Overview

This project allows you to protect sensitive PDF documents by:
- Adding user and owner passwords
- Restricting content extraction
- Securing files from unauthorized access

It’s a simple yet powerful utility for improving document security using Python.

---

## ✨ Features

- 🔑 Add password protection to any PDF
- 🛡️ Restrict copying or extracting content
- ⚙️ Simple and easy-to-use script
- 📂 Works with any standard PDF file

---

## 🧠 Concepts Used

- Python file handling
- Working with external libraries (`pikepdf`)
- PDF encryption and permissions
- Basic cybersecurity principles
